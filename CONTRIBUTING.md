# Contributing

Thanks for your interest in improving **node-backend-starter**. Please follow these guidelines to keep the project consistent and production-ready.

## Development

1. Install dependencies: `npm install`
2. Run linting: `npm run lint`
3. Run tests: `npm test`
4. Run coverage: `npm run coverage`
5. Build the package: `npm run build`

## Code standards

- Use ES Modules only.
- Keep functions small and reusable.
- Validate external inputs.
- Avoid side effects where possible.
- Do not use `console.log` in source files.

## Pull requests

- Ensure linting and tests pass.
- Include a clear description of changes.
- Update documentation when behavior changes.
